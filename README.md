# üçï Pizza Sales Analysis (SQL Project)

##  Project Overview
This project analyzes pizza sales data using **SQL queries** to extract meaningful business insights.  
The goal of this project is to understand sales performance, customer ordering patterns, and revenue trends from pizza sales data.

All insights and results are explained in the accompanying **presentation file** included in this repository.

---

##  Tools & Technologies
- SQL
- Relational Database Management System (MySQL / PostgreSQL)
- Joins, Aggregations, Subqueries
- Date & Time Functions

---

##  Business Questions Solved
The following questions were solved using SQL queries:

1. Retrieve the total number of orders placed  
2. Join relevant tables to find the category-wise distribution of pizzas  
3. Calculate the total revenue generated from pizza sales  
4. Determine the distribution of orders by hour of the day  
5. Identify the highest-priced pizza  
6. Determine the top 3 most ordered pizza types based on revenue  
7. Identify the most common pizza size ordered  
8. Group the orders by date and calculate the average number of pizzas ordered per day  
9. List the top 5 most ordered pizza types along with their quantities  
10. Join the necessary tables to find the total quantity of each pizza category ordered  
11. Determine the top 3 most ordered pizza types based on revenue for each pizza category  
12. Analyze the cumulative revenue generated over time  
13. Calculate the percentage contribution of each pizza type to total revenue
